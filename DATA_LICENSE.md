# Data licensing and citation

The analysis data in `data/raw/` are included to support reproducibility of the paper. The cooked-ham dataset is attributed to Mahieu, Visalli and Schlich (2022) and the open dataset article by Visalli et al. (2024). Please cite both works when using these data.

The MIT license in this repository applies to the analysis code written for this project. Original data, questionnaire materials and third-party bibliographic material remain governed by their original source terms.
